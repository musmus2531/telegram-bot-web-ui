# Telegram Bot Web UI

This project is a web user interface for managing and interacting with Telegram bots. It provides an intuitive and user-friendly way to set up and configure your bots, view logs, and interact with Telegram services.

## Project Setup Instructions

1. **Clone the Repository**
   Clone the repository to your local machine using the following command:
   ```bash
   git clone https://github.com/musmus2531/telegram-bot-web-ui.git
   ```

2. **Navigate to Project Directory**
   Change your directory to the cloned project folder:
   ```bash
   cd telegram-bot-web-ui
   ```

3. **Install Dependencies**
   Use your package manager to install the necessary dependencies. For example, if you're using npm:
   ```bash
   npm install
   ```

4. **Run the Application**
   Start the server by running:
   ```bash
   npm start
   ```

5. **Access the Application**
   Open your web browser and go to `http://localhost:3000` to access the Telegram Bot Web UI.
